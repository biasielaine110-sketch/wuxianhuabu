# 一键迁移包说明（从当前电脑迁移到新电脑）

适用工程路径：`d:\Ai\脚本\无限ai画布---节点连线模式`

## 目标

- 把当前项目完整带到另一台电脑继续开发
- 保留你的画布工程内容（JSON）
- 在新电脑快速跑起来

---

## 第一步：在当前电脑准备迁移包

### 1) 导出你的画布项目 JSON（强烈建议）

在应用里操作：

1. 点击顶部 `项目`
2. 点击 `另存为 JSON`（或 `覆盖保存到上次JSON`）
3. 把导出的 `.json` 放到一个好找的位置（例如桌面）

> 这个 JSON 是你画布内容备份，防止迁移后内容丢失。

### 2) 打包代码目录（推荐 ZIP）

把整个目录打包：

`d:\Ai\脚本\无限ai画布---节点连线模式`

建议压缩成：`无限ai画布-迁移包.zip`

### 3) 可选：减小体积（不影响功能）

打包前可删除：

- `frontend\node_modules`
- `backend\node_modules`
- `frontend\dist`（如果有）

这些到新电脑会重新安装/生成。

---

## 第二步：拷到新电脑

把下面两样拷过去：

1. `无限ai画布-迁移包.zip`（或整个项目文件夹）
2. 你导出的项目 `.json` 文件

---

## 第三步：新电脑初始化

## 1) 安装环境

- 安装 Node.js（建议 LTS，18+）
- 安装完成后打开终端验证：
  - `node -v`
  - `npm -v`

## 2) 解压项目

例如解压到：

`D:\Ai\脚本\无限ai画布---节点连线模式`

## 3) 安装依赖

在项目根目录打开终端，执行：

```bash
cd frontend
npm install
cd ../backend
npm install
```

> 如果你只跑前端，可先只安装 `frontend`。

## 4) 启动项目

优先方式：

- 双击项目根目录下的 `run.bat`

或手动：

```bash
cd frontend
npm run dev
```

---

## 第四步：恢复画布内容

启动后在网页里操作：

1. 点击 `项目`
2. 点击 `导入 JSON`
3. 选择你从旧电脑导出的 `.json`

完成后即可继续编辑。

---

## 常见问题

## 1) 新电脑打开后画布是空的

这是正常的，说明还没导入你的项目 JSON。按“恢复画布内容”步骤导入即可。

## 2) 报依赖相关错误

删除 `node_modules` 后重新安装：

```bash
cd frontend && npm install
cd ../backend && npm install
```

## 3) 端口占用导致无法启动

先关闭旧进程，或者改端口后重启。

---

## 迁移最简版（30秒）

1. 旧电脑先 `另存为 JSON`
2. 拷整个项目文件夹到新电脑
3. 新电脑 `npm install`（frontend/backend）
4. `run.bat` 启动
5. 导入 JSON
